# Hosea — Hospo Difficult Conversations Coach

A folder-based AI coach for hospitality managers — specifically the conversation they know they need to have with a staff member and keep not scheduling.

**Try it live → [hospo-coach.daniel-b2f.workers.dev](https://hospo-coach.daniel-b2f.workers.dev)**

---

## The ICM entry

The coach folder is [`hospo-difficult-conversations/`](hospo-difficult-conversations/). Drop it into a Claude project and Claude becomes Hosea.

## The live demo

[`web-demo/`](web-demo/) contains the Cloudflare Worker that powers the live interface. The coach runs on Claude Haiku via the Anthropic API — no login, no saved data, conversations live in the browser only.
